# Design Portfolio (Ruth-Ann Jackson)

Modern portfolio web app built with React + Vite + Tailwind + React Router.

## Run locally

```bash
npm install
npm run dev
```

## Add your profile photo

Place a JPG at:

`public/profile.png`

The About page will automatically show it. If the file is missing, it falls back to initials.

## Project content

Projects are currently defined in:

`src/content/projects.ts`

Assets are served from:

`public/assets/`

## Build

```bash
npm run build
```
